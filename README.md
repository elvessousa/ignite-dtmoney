## Sobre o projeto

O **dt-money** é um aplicativo simples para controle financeiro, desenvolvido durante o Ignite da Rocketseat.

---

## Tecnologias utilizadas

Esse projeto foi desenvolvido utilizando as seguintes tecnologias abaixo:

- React
- Style Components
- TypeScript
